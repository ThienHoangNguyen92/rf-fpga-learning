# Mô phỏng matching mạng L tại 100 MHz

## 🎯 Mục tiêu
- Ghép trở kháng từ tải 200Ω về nguồn 50Ω tại tần số 100 MHz.

## 🧮 Tham số
- L = 200 nH
- C = 19 pF

## 📊 Kết quả
- SWR đo được ~1.185 tại 100 MHz
- Mô phỏng bằng SimNEC cho thấy điểm Z gần tâm Smith chart.
